# pizzano-MERN

## Setup Backend

### Environment Setup

- create a new file named as <mark> .env </mark>
- for references you can copy <mark>.env.example</mark> file into .env file. Don't forgot to change DB_URL

### Run Setup

- install all dependencies
- ```bash
  $ npm i
  ```
- start server
- ```bash
  $ node -r esm main.js
  ```
